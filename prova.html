Questões de Múltipla Escolha

  1. Qual é o elemento HTML presente somente no HTML5?

    • <figure>
    • <frame>
    • <legend>
    • <label>

  2. Um Técnico Judiciário que utiliza HTML5 está criando um parágrafo em uma página web que tem muitas palavras longas e, para evitar que o navegador quebre linha no lugar errado, utilizou uma tag para especificar onde, em um parágrafo, será aceitável uma quebra de linha. Qual a tag correta para isso?

    • <break>
    • <wbr>
    • <rt>
    • <br>

  3. Um programador está criando o menu principal da página de abertura de um site, que conterá o bloco principal de links de navegação. Considerando os elementos semânticos da linguagem HTML5, estes links devem ficar imediatamente no interior de qual tag?

    • <mark>
    • <progress>
    • <nav>
    • <navigation>

  4. Um programador está criando o menu principal da página de abertura de um site, que conterá o bloco principal de links de navegação. Considerando os elementos semânticos da linguagem HTML5, estes links devem ficar imediatamente no interior de qual tag?

    • <meter>, <audio>, <headergroup>
    • <mark>, <dd>, <bdi>
    • <bdi>, <audio>, <mark>
    • <navigation>, <rd>, <img>

  5. Quais são, no HTML5, os objetos que permitem o armazenamento de dados localmente no cliente?

    • window e document
    • PUT e POST
    • localStorage e sessionStorage
    • session e cookie
